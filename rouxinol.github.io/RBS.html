Programas

Aqui nesta página é possível ter acesso aos programas que permitem a manipulação, visualização e simulação dos espectros obtidos no LAMFI.

RUMP

Programa shareware para sistema Windows, que simula espectros RBS. Versão original no site www.genplot.com

SIMNRA

Programa shareware para sistema Windows, que simula espectros RBS através de uma interface gráfica mais amigável. Versão original no site www.rzg.mpg.de/~mam/

AXIL

Programa livre para sistema DOS, que simula espectros PIXE. Deve ser instalado na raiz do disco C.

WINAXIL

Programa que simula espectros PIXE para Windows (os arquivos .spe devem ser editados e todos os espaços após “:” devem ser suprimidos)

CLARA

Programa para cálculo de fator de correção de alvo grosso para concentrações obtidas através dos fatores de rendimento de alvo fino.

CONVERSOR

Programa que efetua a conversão dos arquivos (.rbs; .spe; .dat; .chn; .sra) e permite também a alteração das informações experimentais salvas. 

TABELA RAIO-X

Programa gráfico no formato da tabela periódica, com as informações básicas de química, além das linhas mais importantes de raio-X para todos os elementos

TENSÃO

Programa que permite calcular a tensão do acelerador para uma determinada energia do feixe, ou o cálculo reverso.

TRIM

Programa que simula a interação de íons na matéria, dando o perfil em profundidade de implantação nos mais diversos materiais.

XCOM

Programa para cálculo de seção de choque de fótons para espalhamento, absorção, produção de pares e atenuação, na faixa de energia de 1 keV até 100 GeV.
